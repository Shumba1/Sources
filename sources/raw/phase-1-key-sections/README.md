# Phase 1 Key Sections Raw Packet

## Purpose

This packet stores the remaining raw or semi-structured inputs that will feed Phase 1 canonical content formation for:

- Repair
- Knowledge
- Products
- Perspectives

## What belongs here

- raw markdown notes
- semi-structured guide drafts
- raw PDFs
- mixed-quality feeder material that is useful but not product-safe as-is

## Rules

1. These files are feeder material, not final doctrine.
2. Do not publish directly from this packet.
3. Prefer `sources/derived/**` when a relevant derived pack already exists.
4. Use this packet when building new dossiers or when derived coverage is incomplete.
5. Any raw claim, frame, or script that conflicts with Tier 1 governance or the voice bible must be transformed, constrained, or discarded during canonical promotion.

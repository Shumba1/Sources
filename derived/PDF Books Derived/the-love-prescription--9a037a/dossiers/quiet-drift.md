## 2. Target fit summary

This source is a strong fit for quiet drift because it describes disconnection as incremental and behavioural rather than dramatic.
The core pattern is not betrayal or explosive conflict. It is parallel lives, logistical talk, missed bids, low play, and fading contact.
The book is especially useful because it translates anti-drift repair into repeatable micro-moves instead of vague advice like “connect more.”
It should not become upbeat fluff, nor should it imply that drift is fixed by one cheerful ritual.
Its real value is in operationalising how to start turning back toward each other.

## 5. Core content logic

- problem pattern: The relationship is not necessarily in open crisis, but it feels flat, logistical, underfed, and less alive than before.
- likely user state/substate: “We are polite roommates”; “We only talk about errands”; “Nothing is terribly wrong, but the warmth is gone.”
- what is happening: connection is being lost through missed bids, low curiosity, low couple time, and low play.
- what makes it worse: assuming meaningful connection requires lots of time; treating busyness as neutral; waiting for the perfect mood; letting every conversation become operational.
- first corrective principle: do not wait for romance to magically reappear—restore micro-contact first.
- first behaviour move: one daily check-in and one visible turning-toward move today.
- script logic: this module should give users plain language for “I think we have gone operational. I want us to turn back toward each other in small ways now.”
- tonight move: protected connection block with no logistics and one open-ended question.
- reset logic: seven days of bids, gratitude, curiosity, touch, and one protected couple-space.
- escalation boundary: if the flatness covers contempt, fear, active secrecy, or repeated refusal of any warmth, branch out.
